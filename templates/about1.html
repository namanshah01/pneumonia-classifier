About

The project

This is a binary image classifier, categorizing chest X-rays of people as 'pneumonic' or 'normal'.
The prediction is done through a ConvNet trained on 5000+ images from a Kaggle Dataset. We used data augmentation, max pooling and batch normalization to reduce the variance problems in the model. Managed to acheive 0.94 recall and 0.91 f1-score.
For detailed info on the model, see this file.

We integrated this into a web app using Flask, a python micro framework. Took the help of this GitHub repo, modified the homepage(index.html) and the main flask file(app.py) to suit the model. Moreover, added another ABOUT page with details of the project.
