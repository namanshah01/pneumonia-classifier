About

The project

This is a binary image classifier, categorizing chest X-rays of people as pneumonic or normal.
The prediction is done through a ConvNet trained on 5000+ images from a Kaggle Dataset. Managed to acheive 0.83 recall, 0.91 precision and 0.87 f1-score. For detailed info on the model, see this folder.

Integrated the model into a web app using Flask. Took this GitHub repo as a reference, and modified and added functionality as per the model. Deployment is done through Heroku.

You can see the souce code uploaded on GitHub
